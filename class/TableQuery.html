Instance Method Details

# (Q.Promise) exec(token, options)

Method executes the current table query object against the supplied model class.

Parameters:

  • currentToken ( Object ) — A continuation token returned by a previous listing operation. Please use 'null' or 'undefined' if this is the first operation.
  • options ( Object ) — The request options.

Options Hash: (options):

  • locationMode ( LocationMode ) — Specifies the location mode used to decide which location the request should be sent to. Please see StorageUtilities.LocationMode for the possible values.
  • timeoutIntervalInMs ( Number ) — The server timeout interval, in milliseconds, to use for the request.
  • payloadFormat ( String ) — The payload format to use for the request.
  • autoResolveProperties ( Boolean ) — If true, guess at all property types.
  • maximumExecutionTimeInMs ( Number ) — The maximum execution time, in milliseconds, across all potential retries, to use when making this request. The maximum execution time interval begins at the time that the client begins building the request. The maximum execution time is checked intermittently while performing requests, and before executing retries.
  • useNagleAlgorithm ( Boolean ) — Determines whether the Nagle algorithm is used; true to use the Nagle algorithm; otherwise, false. The default value is false.
  • entityResolver ( Function ) — The entity resolver. Given a single entity returned by the query, returns a modified object which is added to the entities array.
  • propertyResolver ( TableService.propertyResolver ) — The property resolver. Given the partition key, row key, property name, property value, and the property Edm type if given by the service, returns the Edm type of the property.

Returns:

  • ( Q.Promise ) — resolves with the reponse from the server.
